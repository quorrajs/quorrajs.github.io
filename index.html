Features

QuorraJS is a full fledged, open-source MVC NodeJS framework. Quorra helps to develop web applications using simple and clean code in a short time.

  • Speed

    Engineered with performance tuning in mind. Quorra process http requests at wrap speed.

  • Routing

    Quorra router provides advanced routing capabilities yet its very easy to write routes.

  • REST Support

    Resource controllers make it painless to build RESTful controllers around resources.

  • Security

    Built with security best practices. Ships with built in basic authentication support.
